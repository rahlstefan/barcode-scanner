🚀 БЫСТРО НАЧАТЬ (5 минут):

1. Создайте GitHub репозиторий:
   https://github.com/new

2. Получите Expo Token:
   npm install -g eas-cli
   eas login
   eas credentials

3. Добавьте EXPO_TOKEN в GitHub Secrets:
   Settings → Secrets → New Secret
   Name: EXPO_TOKEN
   Value: <токен-из-eas>

4. Push кода:
   git remote add origin https://github.com/YOUR_USERNAME/barcode-scanner
   git add .
   git commit -m "Initial commit with CI/CD"
   git push -u origin main

5. Смотрите Actions tab:
   GitHub → Actions → Build iOS with EAS

6. Скачайте .ipa:
   https://expo.dev/builds → Download
